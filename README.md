## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Itsmhmod/my-website.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm start
   ```
